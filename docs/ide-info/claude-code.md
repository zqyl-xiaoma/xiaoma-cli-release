# XIAOMA Method - Claude Code Instructions

## Activating Agents

XIAOMA agents are installed as slash commands in `.claude/commands/xiaoma/`.

### How to Use

1. **Type Slash Command**: Start with `/` to see available commands
2. **Select Agent**: Type `/xiaoma-{agent-name}` (e.g., `/xiaoma-dev`)
3. **Execute**: Press Enter to activate that agent persona

### Examples

```
/xiaoma:xmc:agents:dev - Activate development agent
/xiaoma:xmc:agents:architect - Activate architect agent
/xiaoma:xmc:workflows:dev-story - Execute dev-story workflow
```

### Notes

- Commands are autocompleted when you type `/`
- Agent remains active for the conversation
- Start a new conversation to switch agents
